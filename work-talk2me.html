Talk2Me

Talk2Me is an interactive mobile language-learning service for people who want to learn language not in a study-like way but still want it to be instructive enough.


The scope of this project included not only just designing user interface of the application, but also developing concepts and ideas of the service from very fundamental level.

  • CompanyQualson, Inc.
  • Period 2013
  • Scope Mobile App UX/GUI (iPhone, Android)

Improving the existing language-learning experience on mobile

The project was started to develop a new lineup for Qualson, a language learning startup in South Korea. The main objective of this project was to overcome weakpoints of the company's existing service, and to make users utilise their spare time for learning language.

A

Adapt instant messaging-like interface

Reduce burden of users' input process and simplify it.

B

Refresh contents by every 2 hours

Make each content available for limited time only to increase user engagement.

C

Make reviewing better & easier

Highlight corrections and list up corrected answers.

Mobile App UX/GUI

The app is designed to consist of four main tabs; a tab with current question, a tab with previous questions and answers, a tab with user profile, and a tab with miscellaneous settings. Users can check current question and get to the answering screen on the first tab. History tab is designed for reviewing previous questions and answers.
