LINE Kids' Phone

LINE Kids' Phone is a premium smart watch for kids developed by KIWI PLUS. It was launched in the market in late 2015 and distributed KT, one of the major telecommunications services providers in South Korea. The main features include location tracking, calling, and messaging.


Creating the new user experience of wearable device for kids was the main task of UX team. Product design team, web design team, and package design team were also involved in this project.

  • ClientKIWI PLUS
  • Period 2015
  • Scope UX

UX

Designing the user experience for LINE Kids' Phone started with a fundamental question: "What should user experience of kids' wearable device be like?" To emphasise accessibility to each function, we suggested a horizontal widget stream system with minimum depth level.
